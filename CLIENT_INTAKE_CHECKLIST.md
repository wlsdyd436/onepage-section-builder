# CLIENT_INTAKE_CHECKLIST

## 1. 목적

이 문서는 원페이지 제작 전 고객에게 받아야 할 자료를 정리한 체크리스트입니다. 고객과 대화할 때 그대로 체크리스트로 사용할 수 있으며, 자료 누락을 줄이고 제작 속도를 높이기 위한 운영 문서입니다.

## 2. 기본 업체 정보

수집 항목:

- 업체명
- 대표 서비스
- 한 줄 소개
- 주요 지역
- 상담 가능 시간
- 전화번호
- 카카오채널 또는 상담 링크
- 이메일
- 주소
- 사업자 정보
- 로고 파일
- 브랜드 컬러 또는 선호 색감
- 참고 사이트

주의:

- 사업자번호/주소/대표자명 등은 고객이 제공한 정보만 사용합니다.
- 임의로 생성하지 않습니다.

## 3. 원페이지 목표

수집 항목:

- 이번 페이지의 주 목적
  - 상담 문의
  - 전화 유도
  - 카카오 상담 유도
  - 포트폴리오 소개
  - 신규 고객 신뢰 형성
- 가장 강조하고 싶은 서비스
- 타깃 고객
- 피하고 싶은 표현 또는 분위기
- 참고하고 싶은 경쟁사/사이트

## 4. Hero 섹션 자료

수집 항목:

- 대표 문구
- 보조 설명
- 대표 이미지
- 강조하고 싶은 장점 2~3개
- 버튼 문구
- 버튼 연결 대상

예:

- 무료 상담 문의
- 시공 사례 보기
- 전화 상담
- 카카오 상담

## 5. 시공 사례 / Portfolio 자료

각 사례별 수집 항목:

- 사례 제목
- 위치 또는 지역
- 공간 유형
- 평수
- 시공 범위
- 스타일
- 설명 문구
- 대표 이미지
- 사용 허락 여부

권장 수량:

- 최소 3개
- 권장 6개 이상

주의:

- 고객 집 주소가 드러나지 않게 지역은 넓게 표기합니다(예: "서울 강남구"까지만).
- 인물/차량번호/개인정보가 노출되는지 확인합니다.

## 6. 전후 비교 / BeforeAfter 자료

각 항목별 수집:

- 공간명
- before 이미지
- after 이미지
- 위치 또는 공간 유형
- 평수
- 시공 범위
- 변화 설명
- 이미지 사용 허락 여부

주의:

- before/after는 가능하면 같은 각도에서 촬영한 사진을 권장합니다.
- 이미지 비율이 너무 다르면 crop 확인이 필요합니다.
- 실제 고객 공간이면 허락 여부를 반드시 확인합니다.

## 7. 고객 후기 자료

수집 항목:

- 후기 내용
- 고객명 표기 방식
  - 김OO 고객
  - 서울 강남 고객
  - 30평대 아파트 고객
- 시공 유형
- 지역
- 별점 사용 여부
- 후기 사용 허락 여부

주의:

- 실명/전화번호/정확한 주소는 사용하지 않습니다.
- 과장된 성과 보장 문구는 사용하지 않습니다.
- 고객이 직접 제공한 후기만 사용합니다.

## 8. 진행 절차 자료

수집 항목:

- 실제 상담 절차
- 현장 방문 여부
- 견적 방식
- 계약 전 안내 사항
- 시공 진행 방식
- 완료 후 사후 안내 방식

기본 흐름(참고용):

1. 상담 문의
2. 현장 확인
3. 방향 제안
4. 견적 및 일정 안내
5. 시공 진행
6. 완료 및 사후 안내

## 9. FAQ 자료

수집 항목:

- 고객이 자주 묻는 질문
- 답변
- 답변 시 주의할 표현
- 계약/환불/하자보수 관련 안내 여부

주의:

- 법적 책임이 생길 수 있는 단정 표현은 주의합니다.
- "100% 보장", "무조건", "절대" 같은 표현은 피합니다.
- 상세 조건은 "상담 후 안내드립니다"처럼 안전하게 표현합니다.

## 10. 서비스 소개 자료

수집 항목:

- 제공 서비스 목록
- 각 서비스 설명
- 포함 범위
- 제외 범위
- 추천 대상
- 상담 필요 여부

예:

- 아파트 인테리어
- 상가/매장 인테리어
- 주방 리모델링
- 욕실 리모델링
- 부분 시공
- 수납/공간 개선

## 11. 연락/전환 정보

수집 항목:

- 대표 전화번호
- 카카오채널 링크
- 네이버 톡톡 링크
- 인스타그램 링크
- 블로그 링크
- 상담 가능 시간
- 선호 문의 방식
- 긴급 문의 가능 여부

주의:

- `tel:` 링크에 들어갈 전화번호 형식을 확인합니다(예: `010-1234-5678`).
- 외부 링크는 실제 접속이 되는지 확인이 필요합니다.

## 12. 이미지 자료 제출 기준

자세한 파일명/경로/우선순위 규칙은 [IMAGE_ASSET_CHECKLIST.md](./IMAGE_ASSET_CHECKLIST.md)를 함께 참고합니다.

수집 기준:

- 로고
- Hero 이미지
- Portfolio 이미지
- Before 이미지
- After 이미지
- 기타 보조 이미지

파일 기준:

- 가능한 원본 고화질로 받습니다.
- 웹용 최적화는 제작자가 별도로 진행할 수 있습니다.
- 파일명은 영문 소문자를 권장합니다.
- 사용 허락 여부를 확인합니다.
- 개인정보 노출 여부를 확인합니다.

> 3. 로고 파일

> 인물, 주소, 차량번호 등 개인정보가 보이는 이미지는 사전에 알려주세요.

## 16. 현재 프로젝트와 연결

현재 `onepage-section-builder`에서 위 자료들이 실제로 들어가는 위치입니다(`src/data/clients/interior-sample.json` 기준).

- 업체 정보 → `site` (그리고 `footer`/`contact` 관련 필드에서 재사용)
- Hero 자료 → `hero`
- 시공 사례 → `portfolio`
- 전후 비교 → `beforeAfter`
- 고객 후기 → `reviews`
- 진행 절차 → `process`
- FAQ → `faq`
- 서비스 소개 → `services`
- 연락 정보 → `site`(전화/카카오/주소) + `footer`(하단 내비게이션/연락처) + `header.cta`/`StickyMobileCTA01`(전환 CTA)
